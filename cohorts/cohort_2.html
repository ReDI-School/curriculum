Redi School curriculum cohort 2

Aims and goals

  • An ability to use a linux machine with fluency and confidence. This means:
    • Basic command line knowledge 
      ls cd mkdir touch apt-get
      , what 
      sudo
      does
    • User management, ability to install packages through the gui
    • Understanding that linux != ubuntu
  • Understanding how to program. This means:
    • expressions, variable assignments, arithmetic
    • Array comprehensions, inheritance, functions
    • debugging, working out how to do things, googling
  • The ability to create and deploy a complex(?) website. This means:
    • Learning enough HTML and CSS to express their visual sense through code
    • Having a basic understanding of the box model, and HTML nesting
    • Understanding CSS, and maybe a preprocessor (SASS?)
    • Knowing enough git to deploy to github pages, and to solve merge conflicts
